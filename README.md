# 證件圖片浮水印工具

一個純前端證件、文件、申請證明圖片浮水印產生器。使用者可以匯入圖片，設定重複排列嘅文字浮水印，並下載成 PNG 或 JPG。

## 功能列表

- 匯入本機圖片並即時預覽
- 支援點擊選擇圖片同拖放圖片
- 使用 Canvas 將浮水印真實繪入圖片
- 可調整浮水印文字、文字大小、水平間距、垂直間距、透明度、旋轉角度同文字顏色
- 可調整 JPG 輸出品質
- 支援下載 PNG 同 JPG
- 空白浮水印文字會輸出原圖，不會報錯
- 重設設定只會重設浮水印參數，不會清除已匯入圖片
- 桌面、平板、手機都可使用

## 使用方式

1. 直接用瀏覽器開啟 `index.html`。
2. 點擊上傳區選擇圖片，或將圖片拖放到上傳區。
3. 調整浮水印設定，預覽會即時更新。
4. 按「下載 PNG」或「下載 JPG」輸出圖片。

## 如何在本機執行

本專案不需要安裝套件、不需要後端、不需要 build step。

```text
直接雙擊 index.html
```

如想用本機靜態伺服器預覽，也可以在專案目錄執行：

```bash
python3 -m http.server 8000
```

然後開啟：

```text
http://localhost:8000
```

## 支援圖片格式

使用 `<input type="file" accept="image/*">`，支援瀏覽器可解碼嘅主流圖片格式，包括：

- JPG / JPEG
- PNG
- WEBP
- GIF
- BMP
- SVG

如果匯入 animated GIF，輸出會是靜態圖片，不會保留動畫。

## 輸出格式

- PNG：`watermarked-image.png`
- JPG：`watermarked-image.jpg`

輸出圖片會盡量維持原圖解析度，因為 Canvas 實際繪製尺寸會設為原圖寬高。

## 隱私說明

本工具所有圖片處理都在瀏覽器本地完成，不會把圖片上傳到伺服器。

本工具沒有後端、沒有資料庫，亦不會主動發送圖片資料到任何外部服務。
